# VFS İtaliya Slot İzləyici 🇮🇹

## Railway-ə Yükləmə Addımları

1. Bu qovluğu GitHub-a yükləyin (yeni repo açın)
2. railway.app saytına GitHub ilə daxil olun
3. "New Project" → "Deploy from GitHub repo" seçin
4. Bu repo-nu seçin → Deploy edin

## Lokal İşə Salma
```
pip install -r requirements.txt
python vfs_slot_checker.py
```
